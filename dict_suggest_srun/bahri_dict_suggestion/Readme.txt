To run the code

step 1: Please check if the path to Convert_utf_wx is same in you machine, else change the path in the code.
step 2: Place the corpus folder in tmp_anu_dir/tmp
step 3: Please ensure that bahri_dict, bagri_UTF_WX.sh and bahri_lookup.py are in the same location.
step 4: Open the terminal from the location where the above files (mentioned in step 3: are present and type the following:
	sh bahri_UTF_WX.sh <name of the corpus>
	sh bahri_UTF_WX.sh ai1E
This will create a file in each folder named 'bahri_dict_suggestion.txt'
